## Full Description

File Rename Factory is a Windows desktop utility for batch renaming files using a clean rule pipeline.

Drop any files into the workspace, stack rename rules in the order you want them applied, and see the live preview before applying anything. Each rule is fully configurable and the preview flags collisions and Windows-illegal characters before you commit.

**Rules included:**
- Add prefix or suffix
- Auto numbering (001, 002 …) with custom start and digit width
- Insert date (YYYY-MM-DD, YYYYMMDD, YYYY_MM_DD, DD-MM-YYYY)
- Find & replace plain text (case-sensitive or not)
- Regex replace with custom flags
- Case convert — UPPERCASE / lowercase / Title Case / Sentence case
- Remove special characters (with optional keep dots / dashes)
- Trim and collapse whitespace

**Workflow:**
- Live original → new preview table
- Collision and error flags before apply
- Session history with mark-undone
- CSV plan export for review or scripting
- Privacy-first: only file names are read, never file contents

File Rename Factory does not collect data, does not require an account, and does not contain ads.
