## Full Description

PDF OCR Desk is a Windows desktop utility for extracting text from images and scanned PDF files.

Import images or scanned documents, run text extraction, review the results, copy or export the text, and keep a searchable local history — all without an account or internet connection required.

**Features:**
- Import PNG, JPG, JPEG, WebP images
- Import scanned PDF files
- Text extraction workspace with file preview
- Copy extracted text to clipboard
- Export results as TXT or Markdown
- Batch queue for processing multiple files
- Local OCR history with search
- Export center with format options
- Dark and light themes
- Privacy-first: no ads, no tracking, no account required

PDF OCR Desk is a productivity utility. OCR accuracy depends on image clarity, resolution, contrast, and the language of the source document.
