# Product Concept — PDF OCR Desk

## Problem it solves
Windows users who receive scanned documents, photographed invoices, screenshots of text, 
or image-only PDFs have no built-in way to extract the text. PDF OCR Desk provides a clean 
local tool for this without requiring cloud services or subscriptions.

## Target users
- Administrative and office workers processing paper documents
- Students digitising handwritten or printed notes
- Freelancers handling scanned invoices and contracts
- Researchers working with historical or printed materials
- Anyone who receives image-based documents regularly

## Core value loop
1. Import image or PDF
2. Extract text (locally)
3. Copy / edit / export
4. Optional: save to local history for reference

## Differentiators
- No account or subscription required at any tier
- Privacy-first: files never leave the device
- Clean, professional UI — not a web-scraper port
- Batch processing built in from v1.0
- Export in multiple formats (TXT, Markdown, JSON)
