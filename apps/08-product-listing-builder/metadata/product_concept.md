# Product Concept — Product Listing Builder

## Problem it solves
Selling the same product on Amazon, eBay, Etsy, Shopify, and WooCommerce means re-formatting the same content
five times — title-length rules, bullet conventions, tag limits, HTML vs plain. Most listing tools live online,
want subscriptions, and require seller credentials before you can even draft. Product Listing Builder is a clean,
local Windows desktop tool to compose once, preview per marketplace, and export ready-to-paste content.

## Target users
- Indie sellers across Etsy, Shopify, and Amazon
- Dropshippers iterating on titles, bullets, and SEO
- Small e-commerce operations preparing seasonal launches
- Anyone with a Notion table of products who wants a real editor

## Core value loop
1. Set defaults once (currency, platform, condition, business name)
2. Open New Listing — fill in name, price, features, tags, SEO, images
3. Watch per-platform preview render Amazon-style, Etsy-style, Shopify HTML in real time
4. Save to vault → duplicate / re-edit / export TXT / export JSON
5. Bulk CSV import for batch creation

## Differentiators
- Real per-platform formatter for 6 marketplaces, not just "preview text"
- SEO score (0–100) with actionable suggestions
- Bulk CSV importer with preview table
- 3 built-in templates + your own
- Title-length warnings before you hit publish
- Local-only storage; no account, no cloud, no API keys, no telemetry
