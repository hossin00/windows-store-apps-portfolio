# Icon Prompt — Product Listing Builder

**For use with AI image generation tools (DALL-E, Midjourney, Firefly, etc.)**

Create a premium Microsoft Store app icon for "Product Listing Builder".

Style requirements:
- Clean, professional, modern
- Sky-blue gradient accent (#0ea5e9 to #0369a1 — sky-500 to sky-700)
- Show a stylised product card with a small price tag attached, optional bullets next to it
- White / light card on a deep navy background
- Rounded square format (app icon shape)
- No text labels in the icon
- No marketplace logos (Amazon / Etsy / Shopify trademarks) — use generic shopping shapes
- No stars, badges, awards, or rating indicators
- Suitable for Microsoft Store listing (300×300 PNG)

Mood: Crisp, organised, ready-to-publish. Modern e-commerce tool.
