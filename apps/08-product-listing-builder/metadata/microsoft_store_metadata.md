## App Title
Product Listing Builder

## Short Description (max 200 chars)
Compose product listings on Windows for Amazon, eBay, Etsy, Shopify, WooCommerce, or generic shops. Per-platform live preview, SEO scoring, CSV bulk import. Local-only.

## Full Description

Product Listing Builder is a Windows desktop tool for crafting marketplace-ready product listings.

**Editor:**
- Name, description, category (8 options), condition
- Price + compare-at price in 6 currencies (USD, EUR, GBP, MAD, CAD, AUD)
- SKU / barcode and optional business name
- Reorderable feature bullets and comma-separated tag chips
- Image slot grid (placeholder for the desktop build)
- SEO title + meta description with sweet-spot counters
- Platform targeting: Amazon, eBay, Etsy, Shopify, WooCommerce, Generic

**Per-platform live preview:**
- Each marketplace's title-length cap enforced with warnings
- Amazon — bullets-first description block
- eBay — paragraph + condition prefix
- Etsy — story-led description, 13-tag cap
- Shopify / WooCommerce — HTML with feature list
- Generic — clean plaintext

**SEO score 0–100** weighs:
- SEO title length (30–60 chars sweet spot)
- Meta description length (120–160)
- ≥ 3 feature bullets
- ≥ 3 tags
- ≥ 3 image slots

**Workflow:**
- Save listings to a local vault with search, 4 filters (category, platform, condition), and 4 sort modes
- Duplicate, re-edit, export TXT or JSON per listing
- Bulk CSV importer with a preview table and pipe-separated feature / tag cells
- 3 built-in templates (Electronics gadget, Clothing item, Home decor) + your own

**Privacy:**
- Local-only storage; no account, no cloud sync, no marketplace API
- No ads, no telemetry, no tracking

Premium UI with splash screen, 3-slide onboarding, page transitions, skeleton loaders, and an activity chart on the dashboard.
