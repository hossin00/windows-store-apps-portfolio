## Full Description

QR Barcode Studio is a Windows desktop utility for generating real QR codes and 1D barcodes locally.

**QR types:**
- URL — link to any website
- Plain Text — notes, messages, codes
- Wi-Fi — share network SSID, password, security mode
- vCard — share contact info (name, phone, email, organisation)
- Email — pre-filled mailto with subject and body

**Barcode formats:**
- Code 128 — general-purpose ASCII
- EAN-13 — 12 or 13 digit retail
- EAN-8 — 7 or 8 digit short retail
- UPC-A — 11 or 12 digit US retail
- Code 39 — uppercase + digits + a few special characters

**Customisation:**
- QR: custom foreground / background colours, size 128–1024px, error correction L/M/Q/H
- Barcodes: bar width, height, colors, optional readable text below
- Live preview updates as you type
- Export PNG (raster) or SVG (vector)

**Workflow:**
- Batch mode — paste a list of URLs or text and export one PNG per line
- Local history with thumbnail preview, search, re-edit, re-export
- Dark / light / system theme
- Privacy-first: no upload, no account, no tracking

QR codes are produced by the open-source qrcode library; 1D barcodes by JsBarcode. Both run in pure JavaScript locally in your browser.
