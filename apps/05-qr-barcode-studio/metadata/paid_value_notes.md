# Paid Value Notes — QR Barcode Studio

## Current state (v1.0 — free)
- Full QR generator with 5 payload types (URL, Text, Wi-Fi, vCard, Email)
- Full 1D barcode generator with 5 formats (Code 128, EAN-13, EAN-8, UPC-A, Code 39)
- Live preview + PNG / SVG export
- Batch QR per line
- Local history with thumbnails and search
- Privacy-first local-only storage
- Dark / light / system theme

## What justifies a paid price in a future version

### Must-have for paid tier:
- Logo overlay in QR center with auto-sized error correction bumping
- Watermark / branding template (logo + brand colours saved as a preset)
- ZIP download for batch mode instead of N sequential downloads
- More QR payload types: SMS, geolocation, calendar event (vEvent), crypto address
- 2D barcode additions: DataMatrix, PDF417, Aztec

### Value-add features for paid tier:
- Rule presets (save QR styles + payload skeletons by name)
- CSV / Excel import for batch mode with column mapping
- Per-format scan-test mode (camera preview to verify readability)
- Print template builder (Avery label sheet layouts)
- Custom font upload for displayed barcode text
- Tauri native save dialog (write to a chosen folder, not just Downloads)

## Suggested pricing when ready for paid:
- $14.99–$24.99 one-time purchase
- No subscription model
- No ads in paid version (same as free)

## Store strategy:
- Launch free to build user base across QR + barcode users
- Move to paid once logo overlay + batch ZIP + DataMatrix / PDF417 land
- Keep a free version available with v1.0 feature set if possible
