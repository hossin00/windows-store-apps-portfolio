# Paid Value Notes — PDF Master Kit

## Current state (v1.0 — free)
- Six real PDF operations powered by pdf-lib: merge, split, compress, rotate, reorder, extract
- Full UI: dashboard, six tool pages, history, settings, privacy, about, help
- Privacy-first local-only storage
- Dark / light / system theme

## What justifies a paid price in a future version

### Must-have for paid tier:
- Native Windows file save dialog via Tauri (write directly to chosen folder)
- Page thumbnail previews rendered with pdf.js
- Real image-aware compression (JPEG / image down-sampling per page)
- Encrypted PDF support with password prompt
- PDF/A export option

### Value-add features for paid tier:
- Batch operations across multiple PDFs (merge entire folders, split many files)
- Watermark add / remove (basic text or image)
- Page numbering and header / footer insertion
- Bookmark editing
- Form field flattening
- OCR-aware extraction integration (link to PDF OCR Desk if user owns both)

## Suggested pricing when ready for paid:
- $19.99–$29.99 one-time purchase
- No subscription model
- No ads in paid version (same as free)

## Store strategy:
- Launch free to gather reviews and validate core operations
- Move to paid once native file dialogs, real thumbnails, and image-aware compression are in
- Keep a free version available with the v1.0 feature set if possible
