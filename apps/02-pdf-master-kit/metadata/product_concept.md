# Product Concept — PDF Master Kit

## Problem it solves
Windows users frequently need to combine, split, rotate, reorder, or extract pages from PDF documents. 
Most desktop PDF editors are paid suites with steep licensing, and most free options are web-based, 
which means uploading sensitive documents to a third-party server. PDF Master Kit provides a clean, 
local desktop tool for these everyday PDF edits with no upload and no account.

## Target users
- Office workers preparing reports, contracts, and meeting decks
- Freelancers and consultants assembling deliverables from multiple sources
- Students combining notes, lecture slides, and references
- Administrative staff splitting scanned multi-document batches
- Anyone who needs quick, private PDF page operations on Windows

## Core value loop
1. Pick the tool (Merge / Split / Compress / Rotate / Reorder / Extract)
2. Drop one or more PDFs
3. Configure (order, ranges, angles, page selection)
4. Run — get a real, downloaded PDF
5. Operation is recorded in local history for quick reference

## Differentiators
- No account or subscription required
- Privacy-first: files never leave the device
- Real PDF operations powered by pdf-lib (not a placeholder)
- Six core tools in one consistent UI
- Drag-and-drop page reorder built in
- Clean professional design that fits Windows 11
