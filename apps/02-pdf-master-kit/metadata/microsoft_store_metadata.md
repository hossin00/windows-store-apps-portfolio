## Full Description

PDF Master Kit is a Windows desktop utility for everyday PDF editing. Combine multiple PDFs, split a PDF into parts, compress a file, rotate pages, reorder pages by drag-and-drop, and extract specific pages — all without uploading anything or signing in.

**Features:**
- Merge multiple PDFs into a single file, with reorderable input
- Split a PDF by individual pages, custom ranges, or every N pages
- Compress a PDF by re-saving with object streams enabled
- Rotate all pages or per-page (90 / 180 / 270 degrees)
- Reorder pages with drag-and-drop, with optional per-page rotation
- Extract specific pages into a new PDF
- Local operation history with search
- Dark and light themes
- Privacy-first: no ads, no tracking, no account required

PDF Master Kit performs every operation locally using the open-source pdf-lib library. The structural compress tool is not a lossy image compressor — it removes unused objects to produce a smaller, cleaner file.
