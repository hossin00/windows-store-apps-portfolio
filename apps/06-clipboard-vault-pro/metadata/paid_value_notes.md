# Paid Value Notes — Clipboard Vault Pro

## Current state (v1.0 — free)
- Full snippet vault with 5-type auto-detection
- Tags, pin, favourite, collections (many-to-many)
- Search + filter (type / pinned / favourite / collection / date) with 4 sort modes
- Grid and list views in the vault
- Copy counter + "most used" sort
- Local history with TXT and JSON export
- Premium UI: splash, onboarding, page transitions, skeleton loaders, activity chart
- Privacy-first local-only storage; manual capture only

## What justifies a paid price in a future version

### Must-have for paid tier:
- Tauri clipboard-manager auto-capture (opt-in, gated by Privacy mode)
- Encrypted vault — passphrase-locked snippets with at-rest encryption
- Global hotkey to open the vault from anywhere on Windows
- System tray icon with recent snippets
- Cross-device sync (end-to-end encrypted, optional)

### Value-add features for paid tier:
- Smart suggestions — "you might want to save this" based on clipboard patterns
- Pinned snippet shortcuts (Ctrl+Alt+1…9)
- Rich content support (images, formatted text from web)
- AI-assisted titles and tags
- Snippet templates with `{{placeholder}}` substitution
- Custom collections with colours and icons
- Markdown rendering for code/text snippets
- Snippet diff view between versions

## Suggested pricing when ready for paid:
- $14.99–$24.99 one-time purchase
- No subscription model
- No ads in paid version (same as free)

## Store strategy:
- Launch free to gather users and feedback on type detection accuracy
- Move to paid once encrypted vault + auto-capture + global hotkey land
- Keep a free version with the v1.0 feature set if possible
