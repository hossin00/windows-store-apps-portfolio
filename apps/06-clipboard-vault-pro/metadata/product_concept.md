# Product Concept — Clipboard Vault Pro

## Problem it solves
Everyone has a small stash of text they reuse — onboarding emails, terminal commands, addresses, SSH keys,
template paragraphs, sample URLs. Most clipboard managers either capture everything noisily, sync to the
cloud, or sit behind a subscription. Clipboard Vault Pro is a clean, local snippet vault on Windows with
auto type detection, tags, collections, and a fast manual capture flow.

## Target users
- Developers (snippets, terminal one-liners, URLs)
- Customer-support teams (reply templates, FAQ blurbs)
- Salespeople and recruiters (intro emails, follow-ups)
- Students and researchers (citations, source URLs, quotes)
- Anyone with a "miscellaneous" Notes app full of stuff

## Core value loop
1. Paste any text into the Add field
2. The app auto-tags it as URL / email / code / phone / text and stores it locally
3. Optionally add a title, tags, or drop it into a collection
4. Quick-copy from anywhere — vault, search, collection, dashboard recent list

## Differentiators
- Real auto-detection across five content types (with manual override)
- Collections that a snippet can belong to in multiple at once
- Tags + pin + favourite + copy counter + grid/list views
- Search and filter across type / pin / favourite / collection / date / sort
- Export everything as TXT or JSON
- Manual capture only — no silent clipboard scraping
- Local-only storage; no account, no sync, no telemetry
