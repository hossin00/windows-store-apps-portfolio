## Full Description

Clipboard Vault Pro is a Windows desktop utility for saving the text you find yourself retyping all the time.

**What you can save:**
- URLs — auto-tagged when pasted
- Emails — auto-tagged via standard email regex
- Code — auto-tagged when it has function / import / class shape
- Phone numbers — auto-tagged via international and local formats
- Plain text — anything else

**Organisation:**
- Optional title and editable tags per snippet
- Pin items to the top of the vault
- Favourite items for filtering
- Collections (named folders) — snippets can belong to many
- Copy counter tracks reuse, "Most used" sort surfaces top items
- Grid and list views in the vault

**Search & filter:**
- Full-text search across title, content, tags
- Filter chips for type
- Pinned-only / Favourites-only segments
- Filter by collection
- Date range
- Sort by newest, oldest, most used, or pinned first

**Privacy & data:**
- Snippets, collections, tags, and settings live in your browser's localStorage
- No account, no cloud sync, no telemetry, no ads
- Manual capture only — the app never reads your OS clipboard silently
- Export everything as TXT or JSON

Storage cap (100 / 500 / 1000 / unlimited) and auto-detection are tunable in Settings. Dark, light, and system themes.
